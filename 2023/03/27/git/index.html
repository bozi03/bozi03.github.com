ssh公钥

  • 打开命令行输入 ssh-keygen -t ed25519 -C "xxxxx@xxxxx.com" 邮箱地址自己更改.写邮箱是方便辨认.
  • 回车后会提示输入公钥文件名字,文件存放在 C:\Users\用户名\.ssh路径下面, pub后缀名的是公钥,没有后缀名的是私钥.
    • 没有指定文件名的话,文件就是id_ed25519,如果换了其他生成算法应该也是id_算法名的格式
  • 使用cat 文件路径命令读取公钥或私钥
    • cat ~/.ssh/xxx.pub 也可以访问到指定文件名的公钥或私钥
  • 使用ssh -T git@github.com检测github公钥是否部署成功
    • ssh -T git@gitee.com gitee当让是使用这个,换一下网址就好
